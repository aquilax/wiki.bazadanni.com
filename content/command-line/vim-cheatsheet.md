# Vim Cheatsheet

## Shortcuts

Legend:
* N - normal mode

### Moving

N - `w` - move forward one word
N - `b` - move backrwards one word

### Operations

N - `J` - deletes end of line
N - `x` - delete the character

## Commands

`:e!` - reload the original version of the currently opened file

## Help

`CTRL-]` - follow link
`CTRL-T` - go back
